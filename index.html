How each round works
A
Sensor Layer (auto): Teacher picks RED or GREEN and clicks Run. A1 and A2 always match the traffic light perfectly. A3+ have fixed weights (1–5) — higher weight means more likely to match the light.
B
1st Hidden Layer: Each B student sees their 2 A node colours, weighted by personal trust. Roll die, pick face → output GREEN (+1) or RED (−1). After each round, decide which A node was more useful and increase its trust weight.
C
2nd Hidden Layer: Each C student sees their B inputs, rolls die, picks face, outputs GREEN (+1) or RED (−1). Multiplies by weight. Reports value to teacher.
∑
Decision: Σ > 0 → GREEN · Σ < 0 → RED · Σ = 0 → Undecided. If wrong: C weight decreases. If C was consistently wrong but B was consistent, B weight increases next round.
