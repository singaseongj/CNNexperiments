How it works
A
Sensors (auto): Each A node has a reliability weight. Teacher selects light → clicks Run → nodes are assigned RED or GREEN.
B
Layer B — trust weights: Each B student sees 2 A outputs. Round 1: trust is 3+3 (50/50 dice). After each round they adjust trust (e.g. 2+4) based on which A node was more useful.
C
Layer C — same + multiplier: Each C student sees 2 B outputs. Same trust system as B. Output (+1/−1) × weight = value. Weight starts at 20, adjusts ±10 after each round.
∑
Decision: Sum all C values. Σ > 0 → GREEN · Σ < 0 → RED · Σ = 0 → Undecided.
